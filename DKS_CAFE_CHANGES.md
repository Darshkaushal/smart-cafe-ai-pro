# DK's Cafe A1 UI/UX Update

Customer-facing changes included in this version:

- Brand changed to **DK's Cafe**.
- New aesthetic Gen-Z logo using a premium DK monogram.
- Removed customer-facing words like ML, MySQL, full-stack, admin, owner dashboard, and AI.
- Homepage rewritten like a real cafe/startup landing page, not a project feature list.
- Public Owner Login button removed from the main website.
- Location updated to **Jaipur, Rajasthan**.
- Menu, booking, contact, footer, about page, and chat widget text polished for a real cafe experience.
- Admin panel still exists separately on port `3001`, but customers cannot see a direct login option from the public site.

Owner admin access:

```txt
http://localhost:3001
```

Default owner login after seeding:

```txt
Email: owner@dkscafe.in
Password: Owner@12345
```
