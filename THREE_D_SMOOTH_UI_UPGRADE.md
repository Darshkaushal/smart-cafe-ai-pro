# DK's Cafe 3D Smooth UI Upgrade

This update improves the customer-facing website with richer 3D-style visuals and smoother interaction effects while keeping the stack clean and lightweight.

## Added

- Global animated 3D ambience background.
- Enhanced 3D coffee cup hero scene.
- Orbiting cafe mood cards.
- Floating menu product cards.
- 3D hover/lift effects for premium cards.
- Smooth CTA sections for conversion.
- Reduced-motion accessibility support.
- No heavy 3D dependency required; visuals use optimized CSS and React components.

## Updated pages

- Home
- Menu
- Gallery
- Events
- Offers
- Reviews
- Shared layout and global CSS

## Why this is safer

The upgrade avoids heavy WebGL/Three.js dependencies so the site remains fast on mobile and stable during demos. The 3D feel comes from perspective, transform, blur, gradients and motion timing.
